## 項目說明
這是一個Minecraft Java版資源包，重製了`lzh`（文言文）翻譯，覆蓋遊戲本體及一部分模組。Minecraft从1.17.1開始將社區製作的文言譯本加入了遊戲。儘管如此，其翻譯仍然存在不少不夠符合文言風格的地方，整體風格更加偏向一種圖新鮮的獵奇，而非專業的小學成果。因此我創建了本項目，根據個人的口味替代了原本的`lzh`語言包，並覆蓋了日常整合包中的模組。爲專注於打磨譯本，本項目以1.20.1版本爲基礎，不會及時響應遊戲和模組的版本更新。

由於風格、翻譯標準、正字法的差異，本項目暫時獨立運行，不會向現存的官方文言提交更新請求。

目標用戶：
- 好古好文者，欲在Minecraft中體驗古典氛圍的玩家
- 小學（音韻、文字、訓詁）學習者和愛好者
- 文言使用者、跨國筆談者

非目標用戶：
- 好裝腔作勢者，欲以僞文言自詡知古者。不要指望連中學語文課的基礎都沒有就能流利閱讀本譯本，要對一門古典語言有起碼的敬畏。
- 對語言文字不瞭解也不感興趣者

## 翻譯意典
*「意典」者，「理念」之嚴譯也。*

本項目的文言翻譯抱著一種比玩梗和獵奇更加嚴肅的態度。這裡展開講述翻譯的原則和理念。
- **活語言**：與其將文言文視爲一具僵尸而放到解剖臺上研究觀賞，本項目選擇延續文言文，作爲活的書面語。也就是說，本譯本並不存在「用哪個朝代的文言」這個爭議。文言文是漢語的傳承書面語，可以與時俱進，並按需使用所有新老「特性」，就像Minecraft一樣。你放下的石頭是从beta版就有的方塊，而蛙明燈是1.19的新東西。但它們都是同類的Block，遵循同一套底層規則。所以本譯本不會刻意倣古或刻意用古物代新物，也不會故意避開今人更加熟悉的一些表述，而是根據需求合理地取用千年文脈中的原料。佛教譯語和製漢語不會優先使用也不會刻意避開，以準確統一爲本。本譯本既不是先秦文言，也不是宋代文言，也不是桐城體，它是當代文言。
- **書面中心**：文言文是一種目治遠大於耳治的語言，而遊戲內顯示的文本顯然也以視覺而非聽覺的形式傳給玩家的。所以本譯本以書面爲中心，幾乎不攷慮語音。儘管如此，譯本會以**切韻**音係爲假想的語音系統，涉及音譯、同音字的地方都以中古音爲準。
- **參攷多語**：此譯本的底本是`zh_cn`，參攷`en_us`，如有不同，盡量帖近`en_us`的本義。部分詞彙的多種譯法參攷漢字文化圈內的其他譯本，即`zh_tw`、`zh_hk`、`vi_vn`、`ko_kr`、`ja_jp`。選用的詞彙會經過充分的訓詁，攷其字厡語厡、同時參攷英語詞源。不過無需擔心本譯本過度因循英本，畢竟遊戲文本不是經書，不會像翻譯佛經一樣強套梵語語法，也不講什麼「五不翻」。Minecraft對漢語的音譯豁免對此譯本亦有效，不過少數情況會攷慮音譯，如「莈嶄」（modjam，來自[礦藝大典](https://lzh.minecraft.wiki/w/%E8%8E%88%E5%B6%84)，~~此例是現行譯本中少有的能看出譯者有音韻學知識的~~）。
- **正字法**：既然是書面翻譯，正字法也不可不提。不同於使用通行繁體字的內置`lzh`，本譯本採用自製的正字法，如此才算更加「接近古風」。此正字法基於傳承字，在Unicode編碼允許的範圍內修復了一些訛形（如雜-襍、為-爲）和誤用（如烏賊-烏鰂、雕-彫），並恢復了一些字的本字（如繁-䋣、草-艸）。由於編碼的限制，這些修改未必全面類推。與現行繁體字的差異及改動原因詳見[字表]()。譯本內應該不會出現口語文本，所以無需攷慮攷本字的問題。此外，標點也更加古典。逗號「，」均改爲「、」，頓號「、」均改爲「·」，避免使用感歎號和問號，在語气詞後用句號「。」代替。大體類似日文的標點用法。至於直書，就涉及到界面元件的變動，超出一個資源包的能力了。
- **文風分工**：作爲實用文本，譯本不會過於晦澀，也不會在不必要的地方追求所謂的藝術性。務求**明快切近**，沒有白話的啰嗦，流利暢爽，味近古人。遊戲文本有大量的名詞，這些務求精準，而進度、終末之詩等更加靈動的文本會採用文雅一些的譯法，少量用到典故。不刻意規避雙音節詞。不會爲了遷就社區哏而把名字弄得華而不實，如「下界合金」不作「下界玄鐵」作「陰界合金」。
- **遊戲風格**：爲了帖合遊戲風格，一些不存在於Minecraft世界的概念會被更名，即使中英本都沒有這麼做，如「氧化」（「氧」不存在）、「北極熊」（「北極」不存在）、「TNT」（「三硝基甲苯」不存在）。曾經被淘汰的Minecraft譯名如「流髑」也會酌情復用。部分在文言語境中存在歧義的詞也會被更名，如「下界」在大多數文言語境中指相對於上天的人間，且「nether」並無「下」意，故从而改爲「陰界」。又如各類聲音字幕，將統一爲「某物某聲」，避免類似「某物：摔倒」這樣不直觀的格式。
- **統一術語**：術語和專有名詞的翻譯追求系統性，遵循譯名標準化。部分術語使用嚴復的翻譯，還有一些使用我自己的翻譯，會在[譯語詳解](譯語詳解.md)中解釋。所有的語法和詞彙均會給出訓詁理由和參攷的語料，~~順帶批判遊戲內譯本~~。
## 翻譯範圍
本譯本以1.20.1的Minecraft爲基礎，不過翻譯內容包含一些移植高版本內容的模組，所以依然可以看到一部分新東西的譯名。此外，項目包含《終末之詩》的個人譯本，不過需要安裝模組[終末之詩擴展補丁](https://www.mcmod.cn/class/10478.html)才能生效。因爲用不著，`realms`的內容不翻譯。模組的版本對應1.20.1版Minecraft，大多數爲Forge模組。詳見[內容列表](內容列表.md)。
## 如何使用
从Release中下載對應版本的壓縮包，用法和一般的資源包完全一樣。放入目錄加載成功後，在語言列表中選擇文言（華夏）即可。這會覆蓋掉遊戲內置的文言翻譯。

此譯本雖以1.20.1爲基礎，但也可以在更低的版本正常加載。原版遊戲的本地化鍵相對穩定，只增不減，但部分模組可能會出現較大的變動，導致翻譯失效。1.17.1之前的版本在`pack.mcmeta`中以添加`"language"`的形式加入了「文言（華夏）」選項。
